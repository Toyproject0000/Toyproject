요청 url : http://localhost:8080/profile/set

요청형식
var request = http.MultipartRequest('POST', Uri.parse('http://localhost:8080/profile/set'));
request.fields.addAll({
'userId': 'test1',
'info': 'test1',
'nickname': 'test1',
'password': 'test1'
'name': 'test1'
"birth" : "생년월일",
'phoneNumber': 01012345678
'gender': true
});
request.files.add(await http.MultipartFile.fromPath('file', '/C:/Users/kwh87/OneDrive/바탕 화면/히후미.jpg'));

userId만 필수고 나머지는 변경되는 내용만 포함해서 보내주시면 됩니다.

반환 형식
잘되면 ok, 안되면 cancel
